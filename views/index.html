Exercise Tracker

GET all users:

GET /api/exercise/users

GET user's exercise log:

GET /api/exercise/log?{userId}[&from][&to][&limit]

Parameters:

{} required [] optional

from, to dates (yyyy-mm-dd)

limit number

Create a new User

POST /api/exercise/new-user

Add exercises

POST /api/exercise/add
